# E-commerce web en react

Aplicacion web desarrollada con React, MaterialUI, Commerce.js y Stripe.

![Demo aplicativo](./demo.gif)

Para instalar aplicacion ejecutar:

```
npm install
```

Para iniciar aplicacion ejecutar

```
npm start
```
